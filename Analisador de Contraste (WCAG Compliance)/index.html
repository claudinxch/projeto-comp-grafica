Analisador de Contraste WCAG

Verifique se a relação de contraste entre o texto e o fundo está em conformidade com as normas WCAG.

Relação de Contraste:

Como Funciona?

Este analisador calcula a relação de contraste entre a cor do texto e a cor de fundo. A WCAG recomenda uma relação mínima de 4.5:1 para texto normal e 3:1 para texto grande.

Luminância relativa: A luminância de uma cor é a quantidade de luz que ela reflete.

Normas WCAG: Para garantir a acessibilidade, a WCAG exige relações de contraste específicas para melhorar a legibilidade.
